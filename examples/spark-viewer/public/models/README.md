# Model Assets

Large model files are intentionally not committed to Git.

For the integrated app, place browser assets in:

```text
frontend/public/models/
```

For an extracted standalone copy of this example, place browser assets in:

```text
examples/spark-viewer/public/models/
```

```text
mc_aerial_c36_light_75_vq_preview_200k.ply
```
